First Time Setup

    1. Open Maple. If there is a window with one of those check boxes that make it never show up again... I suggest clicking the check box.
    2. At the top menu, choose "Tools -> Options"
    3. Choose the third tab "Interface"
    4. Select the drop down menu with description: "Default format for new worksheets" and choose "Worksheet" instead of the default, "Document"
    5. At the bottom, click "Apply globally"
    6. Close Maple and reopen it

    Here you will find commands to try and tricks to making Maple do your bidding. For those with programming experience, think of Maple as just another programming language. Learn the language and you'll do fine.

    To start, try using simple calculator commands that you might use on a TI-30 or the like. If you need something more complex, use the graphical functions in the left-hand tool bar. If you have specific questions, as always, feel free to contact me.asdf
