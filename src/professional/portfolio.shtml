Consumer Baseline Web Application

While interning with National Information Solutions Cooperative, David worked on a web application to display utility meter prediction data. Java, the Google Web Toolkit and Moxie Highcharts were used to display the predictions while the Java Cache class kept track of all recent queries. A custom class was also created to cache all queries in the browser tab, saving significant communication and processing time during resize events.
