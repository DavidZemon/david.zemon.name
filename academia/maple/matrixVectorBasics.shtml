Vector and Matrix: Vector Basics

Using vectors requires an additional package. If you'd like your vectors to be displayed in any remotely logical form, you'll also need to set the format. Begin all vector problems with the commands:

restart
with(VectorCalculus):
BasisFormat(false):

Great, now we're ready to get started. Format for typing vectors is as follows:

V2d := <x1, y>
V3d := <x2, y2, z2>
CrazyVector := <2x+3, y^2-y>

*Note: If, upon typing the closing bracket of a vector, your brackets to do not change shape (become thinner) then you have typed something wrong in that window.
