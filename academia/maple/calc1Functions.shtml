Calculus 1/Algebra: Functions

Everything math related from calculus forward is in terms of functions. We've got f(t), y(x), x(t), L(s) and all sorts of stuff. Maple has a few ways to deal with such functions. The first, we've already covered. Define a variable as a function: f:=x^2 + blah blah blah. You can then use the subs command to give you f(x). But, who wants to type out so many subs commands?

restart
f:=t->x^2-49
f
f(t)
f(0)
f(7)
solve(f(t))

Using this notation can make your math easier to read, more accurate, and allow for fewer Maple commands. The fewer Maple commands you have, the easier it is to re-read your work. Do take notice, the variable f is still undefined. If you define f(t) as a function, f is still an undefined variable. You must type out f(t) every time you want to use your function.

restart
f(t):=x^2-49
f
f(t)
f(0)
f(7)
solve(f(t))

Pretty simple concept here: you can also define a function by typing f(t):=blah blah blah. Maple will ask if you intend to define a function - simply go with the default choice and hit enter again.
