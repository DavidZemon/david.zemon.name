Miscellaneous: Number Conversions

Maple will also do all of your number conversions, including binary, decimal, octal, hexadecimal, ternary, etc. We will use the convert command with special arguments this time - convert(n, output, r) - where n is the input number you want converted, output is the base (in English) which you would like n converted to, and r is the radix of n.

restart;
convert(1001, decimal, 2);
convert(11010, hex, 2);
convert(5304, binary, 10);
convert("1A3", decimal, 16);

Notice that when converting from a hexadecimal number, you must use quotes. r must also be an integer between 2 and 36.
