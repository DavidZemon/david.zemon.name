Calculus 1/Algebra: Subs

Substituting in values is another one of the most commonly used Maple commands. Who wants to find f(0), f(1), f(2), and f(3) for f(x) = (x^2 + 32x + 56) / (x^3 + 45x + 8) with their calculator? Yea... not me. Try this:

restart
f:=(x^2 + 32x + 56) / (x^3 + 45x + 8)
subs(x=0,f)
subs(x=1,f)
subs(x=2,f);subs(x=3,f)

The subs command also takes two arguments: the variable and its new value, and what function you want to perform this substitution on.

Also take notice of the colon in the last line. The colon is used to tell Maple you're done with a command and ready to start another. In this case, I wanted to type both substitutions on one line. Another common place to use this is when given multiple values in a problem. Let's talk quadratic equations:

restart
f:=a*x^2 + b*x + c
subs({a=1,b=32,c=56},f)
solve(subs({a=1,b=32,c=56},f))
f
a:=1;b:=32;c:=56
solve(f)

Notice first that the substitution of the subs command is not permanent. If values are substituted in one command, they will revert back to their variable name in the following command - be it on the same line or not. Also note that, like the solve command, curly brackets can surround an argument, allowing you to substitute in multiple values at once. A third piece to notice in this section is the use of "nested commands". Nested commands are highly discouraged, as they can make your work very hard to read. Though it is possible to write an entire assignment in one line of code, your teacher or professor would never forgive you (or give you any points for that matter).
