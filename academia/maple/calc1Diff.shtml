Calculus 1/Algebra: Diff

Diff - as in, differentiation. Yea, you'll use this one a lot. Here we go:

restart
y:=x^2-5x+10
dydx:=diff(y,x)

Diff gets two arguments: y -- the function that is to be differentiated, and x -- the variable you are differentiating with respect to.

Diff can also be used in Calc 2 and 3, for partial derivatives.

restart
f:=x^2*y+4*y^2+x
dfdx:=diff(f,x)
dfdy:=diff(f,y)
df:=dfdx+dfdy

At least... I sure hope that's how you do multi-variable Calculus. It's been a while - forgive me.

Taking the second, third, fourth, etc derivative is very easy. There are a few ways to do this:

restart
f:=x^2-5x+10
dfdx:=diff(f,x)
dfdx2:=diff(dfdx,x)

restart
f:=x^2-5x+10
dfdx2:=diff(diff(f,x),x)

restart
f:=x^2-5x+10
dfdx2:=diff(f,x,x)

restart
f:=x^2-5x+10
dfdx2diff(f,x$2)

All of these perform the exact same operations. The third and fourth options are suggested if you don't need the first derivative, and the second option is... well... not suggested ever.

Like most everything in Maple, there is a way to do this without typing the commands - a way that makes it appear on the computer as you would write it on your paper. This does, however, require that you define your function as a function, not a variable.

restart
f(t):=x^2-5x+10
f'(t)
f''(t)
f'''(t)
solve(f'(t)=0,t)
f(0);f(1);f(2)
f'(0);f'(1);f'(2)

You can now see why the use of the variable "D" is highly discouraged. Maple returns derivatives slightly differently than you might expect with this method, but it works all the same. Do not confuese '', two apostrophes, with ", one quotation. Apostrophes are what you want, not quotation marks.
