Getting Started

Maple works via a series of commands, and after every command it returns a result. You can tell Maple to add, subtract, multiply, divide, and so much more. For starters, try this, pressing the 'Enter' key after each line:
*Note: You do not need to include spaces

2 + 2
5^2
10^10
24288/352
1000!
(x + 5)^2

As you can see, your commands are black text, formatted very similarly to the way math is written on paper. The result, or the answer, is then returned in blue, centered on the page, one line below the command. Now, let's say you use have a multi-step process that must be completed, and you wish to show all your work. Let's use the following example:
Solve for x: 4x/57-80=0

f := 4x/57 - 80 = 0
h := f + 80
g := h*57
j := g/4

Notice how the := assigns a function, variable, or number to another variable. You can also use variable names such as 'step1', 'step2', 'answer', 'MapleHatesMe' or many others. Some names not suggested are: 'I' and 'D', or any Maple command (which you will begin to learn in the next section).

Inevitably, a time will come when you want to reuse a variable. To ensure your variables are not carried over from one problem to another, begin every problem with the 'restart' command. Restart clears all variables and closes any packages which you previously opened (you will learn about extra packages later as well).
