Differential Equations: Laplace Transforms

Oh Mr. Laplace. What were you smoking when you thought of this? Well, whatever. Here's how to do Laplace transforms and inverse Laplace transforms nice and easy thanks to Maple.
If the first line confuses you, please see the page regarding loading new packages.

restart;with(inttrans):
f:=cos(2*t)
F:=laplace(f,t,s)
g:=exp(-2*t)*t^2
G:=laplace(g,t,s)

Oh yea, there you have it. A Maple command that takes THREE arguments. The first is of course the function you would like to be transformed, the second is the variable you are transforming, the third is your choice of variable to transform into. In this case, I chose to go from the time domain to the spacial domain. Perhaps you'd rather from variable1 to crackVariable2? Choose them however you please - just make sure that the first one matches the variable in your function.

To take the inverse, it's nearly the same:

invlaplace(F,s,t)
invlaplace(G,s,t)

Simply add "inv" to the beginning for INVerse LAPLACE, and you're all set. Swap the s and the t, since you're going backwards. This may show the answer in a different way than you wrote it down, so try using an "is" statement to make sure you got it right.
