v1.3.4 - 8/17/2011

  • Added tests to the "Homework Trend" graph and retitled apropriately.

v1.3.3 - 6/13/2011

  • The grade no longer appears as a "0%" unless a point value is entered into the line. This allows you to easily switch any of the tables to equally waited assignments with different point values (such as homeworks where some have 20 possible points, others 30, 10, or even 40, but all go into the gradebook with the same weight).
  • Changed file extension from a document to a template.

v1.3.2 - 1/13/2011

  • Finally incorporated hidden cells and formulas which allow the user to enter the weight for a section of the grade which has not yet been completed. If you are told a project will be worth 10% of your grade, but you will not complete said project until the end of the term, that 10% will be ignored until a grade is entered (0% is allowed, but I suggest you aim for something higher).
